# Create Aeronautics for Minecraft Fabric 1.21.1

**Create Aeronautics** is a port of the popular addon for **Minecraft** to the **Fabric** modding platform (game version 1.21.1). The mod adds the ability to build fully functional flying vehicles (airships, airplanes, helicopters) with physics, control mechanics, and integration with Create.

> **Note:** This project is a work-in-progress port. Functionality may differ from the original Forge version.

The ZIP archive contains **two JAR files**:
- `create-aeronautics-1.1.3.jar` — the mod itself
- `fabric-api-0.116.10+1.21.1.jar` — required Fabric API

## 📥 Installation (client & server)

1. Install **Fabric Loader** (minimum version 0.15.0) for Minecraft 1.21.1.  
2. Download the ZIP archive via the **[direct link](https://github.com/SS3rfww/Create-Aeronautics-Fabric-1.21.1/releases/download/Create-Aeronautics/Create.Aeronautics.1.1.3.for.mc1.21.1.zip)**.  
3. Extract the archive and take out both `.jar` files.  
4. Place both JARs into the `.minecraft/mods` folder (for singleplayer) or into the `mods` folder on your server.  
5. Launch the game/server using the Fabric profile.

**Requirements:**  
- Minecraft: 1.21.1  
- Fabric Loader: 0.15.0+  
- Fabric API (already included in the archive)

## ✨ Features (in development)

- 🛩️ Build controllable flying vehicles from any blocks  
- 🧲 Realistic physics: weight, lift, 6‑axis control  
- 🔧 Integration with Create mechanics (rotation, stress, kinetic energy)  
- 🌍 Full multiplayer support (Fabric server)

> **Note:** This is a Fabric port. Some features may be missing or unstable. Report bugs and suggestions in [Issues](https://github.com/SS3rfww/Create-Aeronautics-Fabric-1.21.1/issues).
